# Invest.AI

Invest.AI 是一个基于大语言模型（LLM）的量化投资研究项目。其内容均由大语言模型生成并用于实验目的，不构成任何投资建议。参考的样本源于中国A股、港股、美股市场上的股票和 ETF 基金。

## 参考

- [TradExpert: Revolutionizing Trading with Mixture of Expert LLMs](https://doi.org/10.48550/arXiv.2411.00782)
